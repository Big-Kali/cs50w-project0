HTML Basic Examples

En este capítulo mostraremos algunos ejemplos básicos de HTML. No se preocupe si usamos etiquetas que aún no conoce. Documentos HTML Todos los documentos HTML deben comenzar con una declaración de tipo de documento: !DOCTYPE html>. El documento HTML en sí comienza con html> y termina con /html> . La parte visible del documento HTML está entre body> y /body> .

EXAMPLE

!DOCTYPE html>
html>
body>
h1>My First Heading. /h1>
p>My first paragraph. /p>
/body>
/html>

La Declaración La declaración representa el tipo de documento y ayuda a los navegadores a mostrar correctamente las páginas web.
